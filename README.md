# arcade_usb
DIY arcade button USB controller

## requirements
* Pro Micro module with ATmega32U4, Arduino Leonardo compatible:
  - https://www.amazon.de/gp/product/B01D0OI90U/
* Fritzing to open pcb-project (fzz) file: 
  - http://fritzing.org
